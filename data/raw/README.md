### Description
The dataset contains transactional data for an online retail company, including:
- Invoice numbers
- Product details
- Quantity and pricing
- Customer identifiers
- Transaction dates

### How to Obtain
1. Visit the UCI Machine Learning Repository
2. Download the Online Retail II dataset
3. Load both yearly files into Excel or a database for cleaning and analysis
